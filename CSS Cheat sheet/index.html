Common CSS Properties

Examples

Property Code Description
Background Color background-color Controls the background color of elements.
Background Image background-image Sets the background image of an element.
Font Family font-family Specifies the typeface in a rule set.
Font Size font-size Set text sizes.
Font Weight font-weight Set the weight (boldness) of text.
Foreground Color color Set the foreground text color of an element.
Letter Spacing letter-spacing Set the distance between each character.
Line Height line-height Set the distance between lines of text.
Opacity opacity Control the transparency of an element.
Text Align text-align Set the text alignment of inline contents.
